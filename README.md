# CounterFix
## A ServiceNow Plugin
### Current version: `0.2`
### Browsers supported:
• Chrome `75.0`

• Brave `0.65`

• Firefox `67.0`

### Description:
* Moves the unique record count from the bottom of a list-type report right next to the total record count to save mouse miles
